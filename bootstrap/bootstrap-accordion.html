Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

You can install Bootstrap in several ways: via CDN, by downloading the compiled files, or via package managers like npm. The simplest way is to use the CDN links for CSS and JS in your HTML file.

Bootstrap 5 has several major changes: it no longer depends on jQuery, it has a new offcanvas component, improved grid system, new utilities, updated forms, dropped support for Internet Explorer, and switched to vanilla JavaScript instead of jQuery for its plugins.
